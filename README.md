# Whisper Woods Backend

Whisper Woods Backend is an object-oriented hotel booking system
developed with Python.

## Main Classes

- Room
- Customer
- Booking
- Hotel

## Features

- Room availability management
- Room filtering by type and guest capacity
- Automatic booking-price calculation
- Customer budget checking
- Booking and cancellation
- Reward-points system
- Reward-point discounts
- Maximum 20% loyalty discount
- Booking history
- Logging to `bookings_history.log`
- Unit testing

## Room Types

- Forest Deluxe - Rooms 101-110
- Whisper Suite - Rooms 111-125
- Garden Family Room - Rooms 126-133
